# Интернет-магазин техники Tech group

## Описание проекта
Веб-приложение интернет-магазина техники. Реализован базовый функционал с использованием **Flask**.

### Основной функционал:
* Просмотр товаров

* Пагинация для удобной навигации по каталогу

* Регистрация и авторизация пользователей

* Редактирование данных пользователя

* Избранное продуктов

* Корзина покупок

* Система отзывов

### Cтек технологий:
* Backend: Python, Flask

* ORM: Flask-SQLAlchemy

* База данных: SQLite3

* Шаблоны: Jinja2

* Фронтенд: HTML, CSS, JavaScript (Bootstrap для стилизации)

## Установка и запуск
### Системные требования

* Python 3.8 или выше

* pip-python

### Шаги установки

1. Склонируйте репозиторий
```
git clone https://github.com/kalachanov/practice.git
cd practice
```

2. Создайте виртуальное окружение
```
python -m venv venv
venv\Scripts\activate
```

3. Установите зависимости
```
pip install -r requirements.txt
```

4. (ОПЦИОНАЛЬНО) Создайте базу данных
```
python create_bd.py
```

5. Запустите приложение
```
python app.py
```

6. Откройте в браузере приложение
```
http://127.0.0.1:5000
```
